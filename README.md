## 🎯 About

This repo is a full documentation of a CP journey from scratch to ICPC. It's not just a code dump — every problem solved (from **sheets**, **contests**, or **practice**) is paired with a visual flowchart, and many topics include complete written explanations made from scratch.

**Who is this for?**
- 🟢 Beginners starting their CP journey
- 🟡 Students preparing for ICPC
- 🔵 Anyone who learns better with visual explanations

## 📦 What's Inside

### 📋 Sheets
Problem sheets solved completely. Each problem has:
- A `.cpp` solution
- A `.png` flowchart showing the thought process
- Some sheets also include a **full written explanation** covering the topic from scratch

### 🏅 Contests
Solutions from CP contests. Same format — code + flowchart for every problem.

### 🧩 Practice Problems
Standalone problems solved during practice. All documented the same way.

## 🗺️ How Each Problem is Documented

Every problem folder looks like this:

```
Problem_Name/
├── solution.cpp       ← Clean C++ solution
└── flowchart.png      ← Visual flowchart of the approach
```

For sheets that have a full explanation:

```
Sheet_Name/
├── explanation.pdf    ← Full topic explanation (handmade)
├── Problem_01/
│   ├── solution.cpp
│   └── flowchart.png
├── Problem_02/
│   ├── solution.cpp
│   └── flowchart.png
└── ...
```

## 🗺️ Roadmap

The journey follows a structured path — starting with **Assiut University's CP Sheet**, then moving to advanced books.

---

### Phase 1 — C++ (Assiut Sheet)
Fundamentals of C++ and problem-solving through the Assiut CP sheet.

- Basic I/O & Data Types
- Conditions & Loops
- Functions & Recursion
- Arrays & Strings
- STL (vectors, sets, maps, ...)
- Sorting & Searching
- Greedy Algorithms
- Math & Number Theory

---

### Phase 2 — Data Structures (Assiut Sheet)
Core data structures covered in the Assiut sheet.

- Stacks & Queues
- Linked Lists
- Trees & Binary Search Trees
- Heaps & Priority Queues
- Segment Trees
- Fenwick Tree (BIT)
- Disjoint Set Union (DSU)
- Graphs (BFS / DFS)
- Shortest Paths (Dijkstra, Bellman-Ford)

---

### Phase 3 — Books

After finishing the Assiut sheet, the journey continues with these books:

| Book | Author |
|------|--------|
| 📘 **Competitive Programming 3** | Steven Halim |
| 📗 **Competitive Programmer's Handbook** | Antti Laaksonen |
| 📙 **UNESCO CP Guide** | ICPC Foundation |

## 🤝 Contributing

This repo is open for everyone. If you want to contribute:

- 💡 Found a better solution? Open a PR

```bash

git checkout -b feature/your-contribution
git commit -m "add: solution + flowchart for [problem name]"
git push origin feature/your-contribution
# Then open a Pull Request ✅
```
