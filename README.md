## 🎯 About

This repo is a full documentation of a CP journey from scratch to ICPC. It's not just a code dump — every problem solved (from **sheets**, **contests**, or **practice**) is paired with a visual flowchart, and many topics include complete written explanations made from scratch.

**Who is this for?**
- 🟢 Beginners starting their CP journey
- 🟡 Students preparing for ICPC
- 🔵 Anyone who learns better with visual explanations

## 📦 What's Inside

### 📋 Sheets
Problem sheets solved completely. Each problem has:
- A `.cpp` solution
- A `.png` flowchart showing the thought process
- Some sheets also include a **full written explanation** covering the topic from scratch

### 🏅 Contests
Solutions from CP contests. Same format — code + flowchart for every problem.

### 🧩 Practice Problems
Standalone problems solved during practice. All documented the same way.

## 🗺️ How Each Problem is Documented

Every problem folder looks like this:

```
Problem_Name/
├── solution.cpp       ← Clean C++ solution
└── flowchart.png      ← Visual flowchart of the approach
```

For sheets that have a full explanation:

```
Sheet_Name/
├── explanation.pdf    ← Full topic explanation (handmade)
├── Problem_01/
│   ├── solution.cpp
│   └── flowchart.png
├── Problem_02/
│   ├── solution.cpp
│   └── flowchart.png
└── ...
```

## 🗺️ Roadmap

The journey follows the **Assiut University ICPC Training** curriculum across 3 levels, then dives into advanced CP books.

---

### 🟢 Phase 1 — Newcomers (C++ Basics)

> *Assiut Newcomers Sheet — 10 Weeks*

| # | Topic |
|---|-------|
| Week 1 | Data Types & Conditions |
| Week 2 | Loops |
| Week 3 | Arrays (1D, 2D, Char) · Sorting · Binary Search · Prefix Sum |
| Week 4 | Strings |
| Week 5 | Functions |
| Week 6 | Math & Geometry |
| Week 7 | Recursion |
| Week 8 | General Easy |
| Week 9 | Structures · General Medium |
| Week 10 | General Hard |

---

### 🟡 Phase 2 — Juniors 1 (Intermediate)

> *Assiut Juniors 1 Sheet — 6 Weeks*

| # | Topic |
|---|-------|
| Week 1 | Complexity Analysis |
| Week 2 | STL (String, Stack, Queue, Deque, Priority Queue) |
| Week 3 | STL (Set, Map) · Merge Sort · Quick Sort |
| Week 4 | Binary Search · Two Pointers · Greedy |
| Week 5 | Bit Manipulation · Bitmask |
| Week 6 | Number Theory (Primes, GCD, LCM, Modular Arithmetic, Factorization) |

---

### 🟠 Phase 3 — Juniors 2 (Advanced)

> *Assiut Juniors 2 Sheet — 12 Weeks*

| # | Topic |
|---|-------|
| Week 1 | Recursion Advanced · Backtracking |
| Week 2 | Dynamic Programming — Basics & 0/1 Knapsack |
| Week 3–4 | Graph Representation · BFS · DFS · Trees · Topological Sort |
| Week 5 | Graph — SCC · Articulation Points · Bridges |
| Week 6 | Shortest Paths — Dijkstra · Floyd · BFS variants |
| Week 7 | *(contest week)* |
| Week 8 | DP — Consecutive · Nested · General Ranges |
| Week 9 | DP — Counting · Building Output |
| Week 10 | DP — Digit DP · Bitmask DP |
| Week 11 | *(contest week)* |
| Week 12 | Segment Tree · Sparse Table · Monotonic Queue |

---

## 🤝 Contributing

This repo is open for everyone. If you want to contribute:

- 💡 Found a better solution? Open a PR
- 🐛 Wrong answer or typo? Report an issue
- 📝 Want to add notes or improve explanations? You're welcome

```bash
git clone https://github.com/your-username/competitive_Programming-The-Road-To-ICPC
git checkout -b feature/your-contribution
git commit -m "add: solution + flowchart for [problem name]"
git push origin feature/your-contribution
# Then open a Pull Request ✅
```
